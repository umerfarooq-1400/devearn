How to Host a Free Website on GitHub Pages in 10 Minutes

June 2025 · 8 min read · Free hosting

Want to put your website online without paying a single rupee? GitHub Pages lets you host any HTML website completely free — no monthly fees, no billing, no credit card. Here is exactly how to do it, step by step.

I use this for all my client projects. Your site goes live at yourusername.github.io/sitename and works exactly like any other website.

What You Need

Requirements

A GitHub account (free to create) · An HTML file ready to upload · 10 minutes of your time. That's it.

Step by Step

01
Create a GitHub account

Go to github.com and sign up. It's free. Use any email you have.

02
Create a new repository

Click the + button (top right) → New Repository. Give it a name like mywebsite or devearn. Set it to Public. Click Create.

03
Upload your HTML file

Click "Add file" → "Upload files". Drag your HTML file in. Important: the file must be named exactly index.html — not homepage.html, not website.html. Only index.html works.

04
Commit the file

Scroll down after uploading. Click the green "Commit changes" button. Your file is now saved in the repo.

05
Enable GitHub Pages

Go to Settings (top menu of your repo) → Pages (left sidebar). Under Branch, select main and / (root). Click Save.

06
Wait 2 minutes

GitHub needs 1-3 minutes to build your site. Go to the Actions tab — wait for the green tick to appear.

07
Open your live site

Your site is now live at:

https://yourusername.github.io/reponame

Replace yourusername with your GitHub username and reponame with what you named your repo.

Common Errors and Fixes

Almost everyone hits one of these two problems. Here is how to fix them:

Problem
404 — File not found
Fix
Rename your file to exactly index.html
Problem
Site not updating after changes
Fix
Wait 2-3 minutes and hard refresh (Ctrl + Shift + R)
Problem
Pages option not showing in Settings
Fix
Make sure your repo is set to Public, not Private

What's Next

Your site is live. Now you can share the link anywhere — WhatsApp, Instagram, LinkedIn, client emails. It works exactly like a normal website.

Want a custom domain like yourname.pk instead of the github.io link? You can connect one for around 500-800 PKR per year. I'll cover that in a future post.

Key Takeaway

GitHub Pages = free website hosting forever. No monthly bills. Every web developer should know this. Now you do.
